回到顶部: window属性的3种实现

1.滚动窗口至文档中的特定位置 参数:左上角的像素点的横坐标，纵坐标
window.scroll(0, 0)

2.滚动到文档中的某个坐标 参数:文档中的横轴坐标，纵坐标
window.scrollTo(0,0)

3.在窗口中按指定的距离滚动文档,
X 是水平滚动的偏移量，单位：像素。
Y 是垂直滚动的偏移量，单位：像素。
正数坐标会朝页面的右下方滚动，负数坐标会滚向页面的左上方。
window.scrollBy(-1000, -1000);

回到顶部:element的属性的2种实现

1.多浏览器兼容
document.body.scrollTop ===0 ? document.documentElement.scrollTop =0:document.body.scrollTop=0;

1.浏览器不兼容---实现方式与锚点类似
const header =document.getElementById('header')
header.scrollIntoView();
